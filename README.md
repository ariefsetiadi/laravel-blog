# Laravel Blog

## How to Use

-   Clone this repository

```bash
git clone https://github.com/ariefsetiadi/boilerplate-nodejs.git
```

-   Intall Dependencies

```bash
composer install
```

-   Rename .env.example

```bash
cp .env.example .env
```

-   Run Generate Key

```bash
php artisan key:generate
```

-   Run Storage Link

```bash
php artisan storage:link
```

-   Run Database Migration

```bash
php artisan migrate
```

-   Run Database Seeder

```bash
php artisan db:seed
```

-   For default user login, please check <b>UserSeeder.php</b> file
-   Done
